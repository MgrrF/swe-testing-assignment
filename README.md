# Quick-Calc

## Project Description
Quick-Calc is a simple calculator application that performs basic arithmetic operations: addition, subtraction, multiplication, and division. It also includes a Clear function to reset the calculator. This project demonstrates software engineering testing principles, including unit and integration testing, and uses Git/GitHub for version control.
